# Volkswagen, Audi, Skoda, Seat: платформа MLB-EVO

## Кодировки и сервисные функции, доступные для автомобилей Volkswagen, Audi, Skoda, Seat на платформе MLB-EVO

### Ассистенты

1. **High beam assistance activation**
   - Requires assistance camera
2. **Lane assist: активация ассистента удержания в полосе**
   - Требуется наличие камеры ассистентов
3. **Lane Assist: патч для автомобилей, оснащенных Audi Side Assist**
   - Примените этот патч после активации Lane Assist, если у вас имеется Audi Side Assist
4. **Lane Assist: предел времени бездействия водителя**
5. **Lane Assist: сила вибрации на рулевом колесе**
   - Для блока A5: 8W6907217
6. **Lane Assist: сила вибрации на рулевом колесе**
   - Для блока A5: 8W6907217
7. **Lane Assist: степень поддержки усилителя руля**
